# 健身记录小程序 — 版本路线图

当前版本：**v0.1.0 MVP**（2026-06-30）

---

## v0.2.0 — 体验打磨（估计 2-3 轮对话）

目标：让现有功能更好用

### 训练页
- [ ] 「最近使用」区域：添加动作时优先显示最近用过的动作
- [ ] 自动填充上一组的重量/次数（减少重复输入）
- [ ] 训练中可修改备注
- [ ] 训练完成页：显示本次训练摘要（几个动作、总组数、耗时）

### 首页
- [ ] 显示具体动作名而非仅仅"已训练 N 分钟"
- [ ] 下拉刷新

### 记录页
- [ ] 左滑删除训练记录
- [ ] 按动作名称搜索筛选
- [ ] 显示每条训练的组数

### 个人页
- [ ] 训练天数统计（不是次数，是去过几天）
- [ ] 点击常用动作查看该动作的历史记录

---

## v0.3.0 — 训练模板（估计 2-3 轮对话）

目标：不用每次手动添加动作

### 模板功能
- [ ] 保存当前训练为模板（命名 + 预设动作和组数）
- [ ] 「从模板开始」：开始训练时选择模板，一键载入动作和组数
- [ ] 模板列表页（在「我的」Tab 中）
- [ ] 编辑/删除模板

### 数据结构新增
```
templates（模板）
├── _id
├── name: string          # 模板名称，如"胸背日"
├── exercises: [{exercise_id, name, sets_count}]
├── _openid
└── created_at
```

---

## v0.4.0 — 进度图表（估计 3-4 轮对话）

目标：能看出来有没有进步

### 动作趋势
- [ ] 点击任意动作 → 查看该动作的历史重量/次数变化
- [ ] 折线图显示：日期 vs 最大重量、总次数

### 个人记录
- [ ] 每个动作展示历史最佳（1RM 估算、最大重量、最多次数）
- [ ] 首页「本周 vs 上周」对比

### 技术选型
- 小程序端 Canvas 绘图，或使用 ECharts 小程序版（echarts-for-weixin）

---

## v0.5.0 — 功能扩展（估计 2-3 轮对话）

### 训练增强
- [ ] 支持计时类动作（平板支撑 N 秒）
- [ ] 支持自重动作（不填重量）
- [ ] 训练中途可暂停/恢复计时器
- [ ] 训练备注（整体训练感受）

### 日历视图
- [ ] 记录页增加日历模式
- [ ] 日历上标记有训练的天数

---

## v0.6.0 — 激励系统（估计 2-3 轮对话）

- [ ] 连续训练天数（streak）
- [ ] 每周训练目标（如"每周 4 天"）
- [ ] 达成目标时首页提示
- [ ] 累计里程碑（10 次、50 次、100 次...）

---

## v1.0.0 — 发布就绪

- [ ] 自定义动作（用户自己添加动作）
- [ ] 数据导出（JSON）
- [ ] 图标/启动页设计
- [ ] 微信审核通过并正式上线
